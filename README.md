# AI Testers – NVM

Znajdziesz tutaj wszystkie informacje potrzebne do zainstalowania i korzystania z **NVM** (Node Version Manager). Rozszerzenie tematów znajdziesz w kursie **(AI_Testers vol. 1 – Dodatki Zaawansowane)[https://jaktestowac.pl/course/ait1-bonusy-2/]** z programu **(AI_Testers)[https://aitesters.pl]**.

## Instalacja projektu

### 1. Sklonuj repozytorium

```bash
git clone https://github.com/AI-Testers-pl/ai-testers-nvmv
cd ai-testers-nvmv
```

### 2. Zainstaluj odpowiednią wersję Node.js

```bash
nvm install 24.13.1
nvm use 24.13.1
```

> Projekt zawiera plik `.nvmrc` z wymaganą wersją Node.js – powyższe polecenia automatycznie ją wczytają.

### 3. Zainstaluj zależności

```bash
npm install
```

---

## 🛠️ Polecenia NVM

| Polecenie             | Opis                                            |
| --------------------- | ----------------------------------------------- |
| `nvm -v`              | Sprawdzenie zainstalowanej wersji NVM           |
| `nvm install 24.13.1` | Instalacja konkretnej wersji Node.js            |
| `nvm use 24.13.1`     | Przełączenie się na konkretną wersję Node.js    |
| `nvm ls`              | Lista wszystkich zainstalowanych wersji Node.js |
| `node -v`             | Sprawdzenie aktualnie używanej wersji Node.js   |
| `npm -v`              | Sprawdzenie aktualnie używanej wersji npm       |
| `npm install`         | Instalacja paczek z pliku `package.json`        |
